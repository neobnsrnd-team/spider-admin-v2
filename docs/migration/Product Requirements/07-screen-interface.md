# [연계 관리] 화면 요구사항

## 기관 관리

**화면 목적**: 연계 대상 기관의 정보를 조회, 등록, 수정, 삭제하고, 기관별 게이트웨이 및 핸들러를 관리하는 화면이다.

**접근 경로**: GNB > 연계 관리 > 기관 관리

**화면 유형**: 표준 CRUD 화면 (템플릿 A 참조)

### 화면 레이아웃 구성

| 영역 | 설명 |
|------|------|
| 검색 필터 영역 | 상단, 검색 유형 드롭다운 + 검색어 + 조회 건수 + 조회 버튼 |
| 도구 모음 | 목록 상단 우측, 엑셀 / 출력 버튼 |
| 목록 영역 | 중앙, 기관 목록 테이블 |
| 페이지 네비게이션 | 목록 하단 |
| 버튼 영역 | 목록 하단, 등록 버튼 |

### 검색 / 필터

| 필터 항목명 | 입력 유형 | 필수 여부 | 기본값 | 유효성 조건 |
|-------------|-----------|-----------|--------|-------------|
| 검색 유형 | 드롭다운 (기관 ID / 기관명 / 게이트웨이) | 선택 | 기관 ID | - |
| 검색어 | 텍스트 입력 | 선택 | 빈 값 | - |

### 목록 표시

| 컬럼명 | 데이터 유형 | 정렬 가능 | 비고 |
|--------|-------------|-----------|------|
| 기관 ID | 텍스트 | 가능 | - |
| 기관명 | 텍스트 | 가능 | - |
| 게이트웨이 | 텍스트/버튼 | 가능 | 클릭 시 게이트웨이 관리 팝업 |
| 핸들러 | 텍스트/버튼 | 불가 | 클릭 시 핸들러 관리 팝업 |

- 목록 클릭 시 동작: 기관 수정 팝업 열림
- 기본 정렬 기준: 기관 ID 오름차순
- 페이지당 기본 표시 건수: 20건

### 등록 / 수정 팝업

| 항목명 | 입력 유형 | 필수 여부 | 수정 가능 여부 | 유효성 조건 |
|--------|-----------|-----------|----------------|-------------|
| 기관 ID | 텍스트 입력 | 필수 | 등록 시만 가능 | 영문/숫자, 최대 20자 |
| 기관명 | 텍스트 입력 | 필수 | 가능 | 최대 100자 |
| 기관 설명 | 텍스트 영역 | 선택 | 가능 | 최대 200자 |
| 사용 여부 | 드롭다운 (Y/N) | 필수 | 가능 | 기본값: Y |

### 버튼 및 액션

| 버튼명 | 노출 조건 | 클릭 동작 | 권한 제어 |
|--------|-----------|-----------|-----------|
| 등록 | 항상 노출 | 기관 등록 팝업 열기 | 쓰기 권한 |
| 엑셀 | 항상 노출 | 검색 결과 엑셀 다운로드 | 전체 |
| 출력 | 항상 노출 | 목록 인쇄 | 전체 |
| 저장 (팝업) | 팝업 열림 시 | 기관 정보 저장 | 쓰기 권한 |
| 삭제 (팝업) | 수정 모드 시 | 확인 팝업 후 삭제 | 쓰기 권한 |
| 게이트웨이 (목록) | 목록 행 내 | 게이트웨이 관리 팝업 열기 | 전체 |
| 핸들러 (목록) | 목록 행 내 | 핸들러 관리 팝업 열기 | 전체 |

### 개별 기능 요구사항

- 게이트웨이 버튼 클릭 시 해당 기관에 연결된 게이트웨이 관리 팝업이 열려야 한다.
- 핸들러 버튼 클릭 시 해당 기관의 전문처리핸들러 관리 팝업이 열려야 한다.

### 비기능 요구사항

| 항목 | 요구사항 |
|------|----------|
| 응답 속도 | 목록 조회 결과가 3초 이내에 표시되어야 한다 |

## 코드 맵핑 관리

**화면 목적**: 기관별로 사용하는 코드 값을 표준 코드 그룹과 맵핑하여 관리하는 화면이다.

**접근 경로**: GNB > 연계 관리 > 코드 맵핑 관리

**화면 유형**: 표준 CRUD 화면 (템플릿 A 참조)

### 화면 레이아웃 구성

| 영역 | 설명 |
|------|------|
| 검색 필터 영역 | 상단, 기관 드롭다운 + 코드 그룹 드롭다운 + 조회 건수 + 조회 버튼 |
| 도구 모음 | 목록 상단 우측, 엑셀 / 출력 버튼 |
| 목록 영역 | 중앙, 코드 맵핑 목록 테이블 |
| 페이지 네비게이션 | 목록 하단 |
| 버튼 영역 | 목록 하단, 등록 버튼 |

### 검색 / 필터

| 필터 항목명 | 입력 유형 | 필수 여부 | 기본값 | 유효성 조건 |
|-------------|-----------|-----------|--------|-------------|
| 기관 | 드롭다운 (전체 + 기관 목록) | 선택 | 전체 | API에서 동적 로드 |
| 코드 그룹 | 드롭다운 (전체 + 코드 그룹 목록) | 선택 | 전체 | API에서 동적 로드 |

### 목록 표시

| 컬럼명 | 데이터 유형 | 정렬 가능 | 비고 |
|--------|-------------|-----------|------|
| 기관 ID | 텍스트 | 가능 | - |
| 코드 그룹 ID | 텍스트 | 가능 | - |
| 코드 | 텍스트 | 가능 | - |
| 코드명 | 텍스트 | 가능 | - |
| 사용 여부 | 상태 아이콘 | 가능 | Y/N |
| 정렬 순서 | 숫자 | 가능 | - |

- 목록 클릭 시 동작: 코드 맵핑 수정 팝업 열림
- 기본 정렬 기준: 기관 ID > 코드 그룹 ID 오름차순
- 페이지당 기본 표시 건수: 20건

### 등록 / 수정 팝업

| 항목명 | 입력 유형 | 필수 여부 | 수정 가능 여부 | 유효성 조건 |
|--------|-----------|-----------|----------------|-------------|
| 기관 | 드롭다운 | 필수 | 등록 시만 가능 | 기관 목록에서 선택 |
| 코드 그룹 | 드롭다운 | 필수 | 등록 시만 가능 | 코드 그룹 목록에서 선택 |
| 코드 | 텍스트 (읽기 전용) | - | 불가 | 코드 그룹 선택 시 자동 표시 |
| 코드명 | 텍스트 (읽기 전용) | - | 불가 | 코드 그룹 선택 시 자동 표시 |
| 맵핑 코드 | 텍스트 입력 | 필수 | 가능 | 최대 50자 |
| 사용 여부 | 드롭다운 (Y/N) | 필수 | 가능 | 기본값: Y |
| 정렬 순서 | 숫자 입력 | 선택 | 가능 | 0 이상 |

### 버튼 및 액션

| 버튼명 | 노출 조건 | 클릭 동작 | 권한 제어 |
|--------|-----------|-----------|-----------|
| 등록 | 항상 노출 | 코드 맵핑 등록 팝업 열기 | 쓰기 권한 |
| 엑셀 | 항상 노출 | 검색 결과 엑셀 다운로드 | 전체 |
| 출력 | 항상 노출 | 목록 인쇄 | 전체 |
| 저장 (팝업) | 팝업 열림 시 | 코드 맵핑 저장 | 쓰기 권한 |
| 일괄 저장 (팝업) | 팝업 열림 시 | 여러 코드 맵핑 일괄 저장 | 쓰기 권한 |
| 삭제 (팝업) | 수정 모드 시 | 확인 팝업 후 삭제 | 쓰기 권한 |

### 개별 기능 요구사항

- 등록 팝업에서 기관과 코드 그룹을 선택하면 해당 그룹의 코드 목록이 표시되어야 한다.
- 코드별 사용 여부와 정렬 순서를 개별 설정할 수 있어야 한다.
- 일괄 저장 기능으로 여러 코드를 한 번에 저장할 수 있어야 한다.

### 비기능 요구사항

| 항목 | 요구사항 |
|------|----------|
| 응답 속도 | 목록 조회 결과가 3초 이내에 표시되어야 한다 |

## 게이트웨이 관리

**화면 목적**: 통신 게이트웨이를 등록/관리하고, 게이트웨이에 연결된 기관을 설정하는 화면이다.

**접근 경로**: GNB > 연계 관리 > 게이트웨이 관리

**화면 유형**: 표준 CRUD 화면 (템플릿 A 참조)

### 화면 레이아웃 구성

| 영역 | 설명 |
|------|------|
| 검색 필터 영역 | 상단, 검색 유형 + 검색어 + 어댑터/리스너 필터 + 조회 건수 + 조회 버튼 |
| 도구 모음 | 목록 상단 우측, 엑셀 / 출력 버튼 |

### 검색 / 필터

| 검색어 | 텍스트 입력 | 선택 | 빈 값 | - |
| 어댑터/리스너 | 드롭다운 (전체 / 어댑터 / 리스너) | 선택 | 전체 | - |

| 게이트웨이 설명 | 텍스트 | 가능 | - | - |

| 어댑터/리스너 | 텍스트 | 가능 | 120px | - |

| 관련 기관 | 버튼 | 불가 | - | 클릭 시 기관 관리 팝업 |
