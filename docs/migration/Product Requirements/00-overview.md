## 전체 화면 목록 (46개)

> 각 화면은 표준 화면 템플릿(A/B/C/D/E)으로 분류됩니다. 템플릿 상세는 `01-common-layout.md` 4장을 참조하세요.
> - **템플릿 A**: 표준 CRUD 화면 (검색 + 목록 + 등록/수정 팝업)
> - **템플릿 B**: 조회 전용 화면 (검색 + 목록, 등록/수정 없음)
> - **템플릿 C**: 모니터링 화면 (자동 갱신, 상태 시각화)
> - **템플릿 D**: 특수 화면 (개별 레이아웃)
> - **템플릿 E**: 목록-상세 전환형 화면 (목록↔상세를 같은 영역에서 전환)

### [인증] (2개)

| 화면명 | 템플릿 | 설명 |
|--------|--------|------|
| 로그인 | D - 단일 카드 | 시스템 접속을 위한 사용자 인증 화면 |
| 개인정보 수정 | D - 단일 폼 | 로그인한 사용자 본인의 프로필 정보를 조회하고 수정하는 화면 |
